## Bug 1: Price Fallback to 1.0 Silently Corrupts Sizing [CRITICAL]

**File:** `pmacs/nervous/orchestrator.py` line 1168
**Risk:** A $50 stock with `current_price=1.0` computes ~5000 shares instead of ~100. Stop set at $0.85 instead of $42.50. Oversized positions violate max 20% capital rule.

**Current code (line 1168-1182):**
```python
current_price: float = 1.0  # fallback default
price_cache = self._get_price_cache()
if price_cache is not None:
    fetched = price_cache.get_price(ticker, cycle_id)
    if fetched is not None:
        current_price = fetched
    else:
        log_debug("PRICE_FALLBACK_DEFAULT", ...)  # warns but continues
```

**Fix:** Abort the symbol when price is unavailable. Do NOT continue with a fallback price.

```python
current_price: float | None = None
price_cache = self._get_price_cache()
if price_cache is not None:
    current_price = price_cache.get_price(ticker, cycle_id)

if current_price is None or current_price <= 0:
    log_debug("PRICE_UNAVAILABLE_ABORT", ...)
    self._symbol_results[ticker] = SymbolResult(
        ticker=ticker, status="ABORTED", abort_reason="PRICE_UNAVAILABLE",
    )
    return
```

**Test:** Add test case to `tests/integration/test_full_cycle.py` or `tests/unit/`:
- Mock price cache returning `None`
- Verify symbol is aborted, not processed with price=1.0
- Verify `ABORTED` status with reason `PRICE_UNAVAILABLE`

## Bug 2: AuditWriter Not Thread-Safe [CRITICAL]

**File:** `pmacs/storage/audit.py`
**Risk:** Non-Negotiable #3 violation. If two `AuditWriter` instances are created concurrently (e.g., re-eval loop + main thread), both read the same `_prev_sha` from file, produce duplicate chain links, breaking hash chain integrity.

**Current code:** `__init__` calls `_recover_last_sha()` (reads entire file). `append()` writes with that stale `_prev_sha`. No locking.

**Fix:** Make `append()` atomically read-last-sha-then-write using `fcntl.flock`.

```python
import fcntl

class AuditWriter:
    def __init__(self, path: str | Path):
        self._path = Path(path)
        self._path.parent.mkdir(parents=True, exist_ok=True)
        self._prev_sha: str = AUDIT_GENESIS_PREV_SHA
        self._fd = None
        if self._path.exists():
            self._recover_last_sha()

    def _recover_last_sha(self) -> None:
        """Scan existing file to recover the last SHA."""
        last_sha = AUDIT_GENESIS_PREV_SHA
        with open(self._path) as f:
            for line in f:
                line = line.strip()
                if line:
                    parts = line.split("\t")
                    if len(parts) >= 5:
                        last_sha = parts[4]
        self._prev_sha = last_sha

    def _open(self):
        if self._fd is None:
            self._fd = open(self._path, "a")
        return self._fd

    def append(self, event_type: str, payload: dict, cycle_id: str = "") -> str:
        """Append an event. Thread-safe via file lock."""
        iso_ts = datetime.now(timezone.utc).isoformat()
        if cycle_id and "cycle_id" not in payload:
            payload = {**payload, "cycle_id": cycle_id}
        canon = canonical_json(payload)

        fd = self._open()

        # Lock, recover latest SHA, write, unlock
        fcntl.flock(fd.fileno(), fcntl.LOCK_EX)
        try:
            # Re-read last SHA from file (another writer may have appended)
            if self._path.exists() and self._path.stat().st_size > 0:
                with open(self._path) as rf:
                    last_sha = AUDIT_GENESIS_PREV_SHA
                    for line in rf:
                        line = line.strip()
                        if line:
                            parts = line.split("\t")
                            if len(parts) >= 5:
                                last_sha = parts[4]
                    self._prev_sha = last_sha

            hasher = hashlib.sha256()
            hasher.update(iso_ts.encode("utf-8") + b"\x00")
            hasher.update(self._prev_sha.encode("utf-8") + b"\x00")
            hasher.update(event_type.encode("utf-8") + b"\x00")
            hasher.update(canon.encode("utf-8"))
            this_sha = hasher.hexdigest()

            line = f"{iso_ts}\t{self._prev_sha}\t{event_type}\t{canon}\t{this_sha}\n"
            fd.write(line)
            fd.flush()
            os.fsync(fd.fileno())
            self._prev_sha = this_sha
        finally:
            fcntl.flock(fd.fileno(), fcntl.LOCK_UN)

        return this_sha

    def close(self) -> None:
        if self._fd is not None:
            self._fd.close()
            self._fd = None
```

**Key changes:**
1. `fcntl.flock(LOCK_EX)` before read-then-write in `append()`
2. Re-read last SHA from file inside the lock (handles concurrent writers)
3. Write while still holding the lock
4. Release lock in `finally`

## Bug 3: Deterministic Paper Signing Key [CRITICAL]

**File:** `pmacs/nervous/orchestrator.py` lines 1863-1867
**Risk:** `hashlib.sha256(b"pmacs_paper_mode").digest()[:32]` is forgeable by anyone reading the source. Violates spirit of Non-Negotiable #1.

**Current code:**
```python
signing_key_hex = self._config.get("execution_signing_key")
if signing_key_hex:
    private_key = bytes.fromhex(signing_key_hex)
else:
    private_key = hashlib.sha256(b"pmacs_paper_mode").digest()[:32]
```

**Fix:** Generate a random key at first run and store in Keychain. Fall back to deterministic key ONLY in test mode.

```python
signing_key_hex = self._config.get("execution_signing_key")
if signing_key_hex:
    private_key = bytes.fromhex(signing_key_hex)
else:
    # Try Keychain first (paper signing key generated at wizard install)
    try:
        from pmacs.storage.keychain import read_key
        stored = read_key("pmacs.execution.signing_key")
        if stored:
            private_key = bytes.fromhex(stored)
        else:
            raise ValueError("No signing key in Keychain")
    except (ValueError, Exception):
        # Test/dev fallback — NOT for production
        import os as _os
        if _os.environ.get("PMACS_TEST_MODE"):
            private_key = hashlib.sha256(b"pmacs_paper_mode").digest()[:32]
        else:
            # Generate and store a random key
            from pmacs.execution.signing import generate_keypair
            priv_bytes, _ = generate_keypair()
            private_key = priv_bytes
            try:
                from pmacs.storage.keychain import set_api_key
                set_api_key("pmacs.execution", "signing_key", private_key.hex())
            except Exception:
                pass  # Best effort — Keychain may be unavailable in CI
```

**Also add to wizard** (`pmacs/installer/wizard.py`):
- During step 4 (Keychain setup), generate and store `pmacs.execution.signing_key`

**Test:** Add test to `tests/unit/test_execution_service.py`:
- Verify key is NOT the deterministic `sha256(b"pmacs_paper_mode")` hash in non-test mode
- Verify `PMACS_TEST_MODE=1` falls back to deterministic key
- Verify Keychain-stored key is used when present
